# expense_accounting
**stack:** `C#`, `WinForms`, `.NET 4.8`

#### small program for adding and tracking your expenses <br> _________________________________________________________________________________________________________________ <br> <br> allows user to add a title, description, cost and category to their expense <br> expenses are displayed in a table implemented using **GridView** <br> the total cost of expenses, as well as individual totals by category, are shown at the bottom of the form <br> exception handling is included <br> _________________________________________________________________________________________________________________ <br>

## getting started

### #1 git clone option

```bash
git clone https://github.com/morphineclipse/expense_accounting.git
cd expense_accounting
```
#### run /bin/Debug/kzprs.exe

### #2 download option

#### click on "code" > "download ZIP" <br> excract downloaded zip <br> run /bin/Debug/kzprs.exe
